INSTALLATION
------------

1. Copy the file Succinct.colloquyStyle file to this folder in your home folder:
     Library -> Application Support -> Colloquy -> Styles

2. Start Colloquy. Or (if already open) type "/reload style" in a Colloquy chat.

3. Select the "Succinct" option from the Style drop-down. Choose the "Appearance
   Preferencesâ€¦" option to set Succinct as your default Colloquy chat style.
